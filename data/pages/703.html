Google Docs Known Issues

If your problem is listed below, we're aware of the issue and are working to fix it as soon as possible. To see more information on an issue, just click on it. Clicking the "Report" button will ensure that we're notified about your issue.

Although this page was originally written in English, you can use Google's website translator to instantly and automatically translate this page:

Commonly reported issues in Google Documents:

  • Images with dimensions greater than 2000 pixels cannot be uploaded
    We're aware of this issue. In the meantime, one workaround is to resize images before you upload them.

    Having this issue?    

  • Certain keyboard shortcuts in Safari don't work
    The keyboard shortcuts for bolding and italicizing text do not work in documents when using Safari. This issue requires a fix in Safari. The fix has been submitted but won’t roll out until the next Safari update. In the meantime, you can use a different browser or a nightly version of Safari.

    Having this issue?    

  • Footnotes may not be visible on screen
    Footnotes in Google Documents may be partially or fully off-screen to the left -- without a way to scroll your browser to view them. This can often be corrected by expanding the size of your browser window or adjusting your screen resolution. We understand that this can be a frustrating experience and apologize for the inconvenience.

    Having this issue?    

  • "Create a Document" Gmail lab may not work for Google Apps users
    Google Apps users working with the "Create a Document" Gmail lab may not be able to create a Google document with the lab. The lab, however, should work fine for non-Apps Gmail users. We apologize for the inconvenience.

    Having this issue?    

Commonly reported issues in Google Spreadsheets:

  • More than one form editing window open overwrites changes saved
    We're aware of a problem that occurs when someone has more than one form editing window open for the same form. The workaround to avoid edits from one form editing window overwriting edits from another is to ensure that you only have one form editing window open. This is the window that opens when you select Form > Edit form.

    Having this issue?    

  • Insert Date/Insert Time shortcuts don't work in Chrome
    We're actively working on fixing this issue. Thanks for your patience.

    Having this issue?    

  • Small number of snippets from publishing charts are broken
    A small number of scripts produced soon after we launched the new version of Charts in Google spreadsheets may be broken. There is an easy resolution if one of your published charts now appears broken. Under the Chart menu in the upper left of the chart, select 'Publish chart.' Copy and paste the new script to publish your chart to a webpage.

    Having this issue?    

  • Sheet protection is not allowing users to set protection to multiple collaborators
    If you are setting sheet protection for certain sheets in your spreadsheet, it will only allow you to set the 'Only me' option for the sheet. We are actively working on a fix to allow you to give access to a particular sheet to specified collaborators using the 'Me, and the collaborators selected below' option.

    Having this issue?    

Commonly reported issues in the Docs List and across Docs:

  • Google Docs Viewer has limited functionality in Internet Explorer
    The File and View menus on the Google Docs Viewer are not currently available in Internet Explorer

    Having this issue?    

  • Sharing folders with "anyone with the link"

    We have temporarily disabled the ability to share folders with the visibility setting "Anyone with the Link". You'll notice it is now grayed out in the sharing settings dialogue.

    When folders are shared under this visibility setting, the included files in the folders are not given the same access rights. So if the files within the folder are set to "private", users who are shared the folder via the link will not be able to access the documents within. A workaround is to either set the folder to public if the contents are not sensitive, or if the contents are sensitive, share the folder with each person who needs access individually.

    Having this issue?    

  • Cannot delete items in bulk from the trash
    At the moment, you cannot bulk delete your trashed items if your trash contains a large number of items. In the meantime, you can select about 50 at a time and empty those from your trash.

    Having this issue?    

Commonly reported issues in Google Drawings:

  • No connectors in Google drawings
    We know that the ability to join objects via a straight or bent line is a top feature request and are working on it.

    Having this issue?
