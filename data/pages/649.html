Refine your image searches: Image search results options

If you want to filter and customize the image search results you see, use the options along the side of the results page. For instance, you can choose to just see black and white photos of a certain size. Use the links below to learn about search tools specific to Google Images.

For more details on how to use search options for Google Images and other search properties, visit this search results options article.

Size

Filter image results by their size (click Any size at any time to get back to unfiltered results):

Medium / Large / Icon

Use these options if you want to see results for a specific range of sizes. Click Medium to find images with resolutions between 400 x 300 pixels and 1024 x 768 pixels. Click Large to find images with resolutions bigger than 1024 x 768 pixels. Click Icon to find square images with the following resolutions: 50 x 50, 64 x 64, 96 x 96, 128 x 128, and 256 x 256. It will also show you images with widths and heights less than or equal to 50 pixels.

Larger than

To find images that are larger than a certain size, use this option. Select the minimum size for your image in the drop-down.

Exactly

If you only want images with a certain size, use this option. Enter the width and height you're looking for, then click Search.

Type

The second group of links filters results by their type. Google Images can automatically detect whether an image is a face, a photograph, clip art, or a line drawing. Click Any type at any time to get back to unfiltered results.

Color

The last group of links filters results by color (click Any color at any time to get back to unfiltered results):

Full color / Black and white

Use these options to see images that contain either a full spectrum of color or are in black and white.

Specific color

Click a colored square under the Specific color link to find images containing that particular color.

We hope these options help you explore and interact with your image search results in useful ways. As we develop new ways to help you search, watch for new or changed options.
