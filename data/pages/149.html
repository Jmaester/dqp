DiggLite

An open-source, OAuth application to showcase Digg's new api. It can serve as a reference application and as a blueprint for any basic Digg-based application. It includes a river of the most recently popular stories, which can be Dugg and buried remotely, and the means of breaking things down by top-level topics. You can find it at digglite.com or download the source code on Github and view the documentation.
