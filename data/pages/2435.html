Watch a video on Google Maps API and Tripadvisor.

The Google Maps API is available for any site that is free for consumer use. You can learn more about the Google Maps API at:

http://code.google.com/apis/maps/index.html

Documentation, FAQs, success stories, additional support, and the Terms of Use are also available.

If you wish to request permission to use Google Maps in other media, please see:

http://www.google.com/permissions/

If you operate an enterprise or commercial website, you may also be interested in Google Maps API Premier.

For more detailed information on how to add a Google Map to your site, visit http://maps.google.com/help/maps/getmaps
