Getting started: Keyboard shortcuts

Use keyboard shortcuts to move around an image results page like a pro! For example, press the space bar on your keyboard to skip from one page of images to the next. Here's a list of keyboard shortcuts that can be useful when you're using Google Images.

Quickly scroll through results:

Function Shortcut keys
Next page Space or Page down
Previous page Shift + Space or Page up
Next row Down arrow
Previous row Up arrow
Top Home
Bottom End

Preview and navigate to results:

Function Shortcut keys
Open hover-over pane for next image Right arrow
Open hover-over pane for previous image Left arrow
Close hover-over pane Escape
Follow link Enter
Select next link Tab
Select previous link Shift + Tab
