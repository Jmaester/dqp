Getting someones IP address isn't sneaky - it can be very useful and
completely above board - and widely used by many websites.

I do it on a site I code and do it within PHP. I do it because,
combining that with some javscript, I want to know a rough profile of
those accessing the site ... what OS they're using, whether they have
the GE plugin installed and whether their platform supports the GE
plugin at all. To record this info I use their IP address as the
"key". (I can then also see whether a user who doesn't have the Plugin
installed returns having installed it. Nothing sneaky and very easy to
do).

in PHP
$ip = getenv('REMOTE_ADDR');
will get the users IP Address

There are then servives you can use to get a location from an IP
Address. Search for something like
"ip address locator service" or the like.

I don't use these myself so have no experience of them but they are
widely used. (Going to Bing Maps for the first time opened the map
with my house at the center I seem to remember - so they do work if no
proxies etc are in the way)

Regards
Nymor
